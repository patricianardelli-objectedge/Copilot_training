# Playwright JavaScript Project

Scaffolded Playwright project (JavaScript)

## Quickstart

Install dependencies and browsers, then run tests:

```bash
npm install
npx playwright install
npm test
```

Files created:

- [package.json](package.json)
- [playwright.config.js](playwright.config.js)
- [tests/example.spec.js](tests/example.spec.js)
- [.gitignore](.gitignore)
